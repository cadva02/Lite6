# Lite6

Proyecto de robótica basado en el brazo robótico **UFactory Lite 6**.

## ¿Qué es?

El Lite 6 es un brazo robótico de 6 grados de libertad (DOF) compacto y ligero, diseñado para tareas de automatización, investigación y educación.

## Características principales

- 6 grados de libertad
- Carga útil: 0.6 kg
- Alcance: 440 mm
- Control mediante ROS 2 y Python SDK

## Requisitos

- ROS 2 (Humble o superior)
- Python 3.8+
- xArm Python SDK

## Uso rápido

```bash
# Clonar el repositorio
git clone https://github.com/cadva02/Lite6.git
cd Lite6

# Instalar dependencias
pip install xarm-python-sdk
```
